## Number Guessing Game (Python)

**Description:**  
A beginner-friendly Python game where the player guesses a random number within a limited number of attempts.

**Concepts Practiced:**
- Variables
- Loops
- Conditionals
- Input and output
- Random number generation

**What I Learned:**
- How to control program flow using conditionals
- How to validate user input
- How to track attempts using loops
